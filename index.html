How to Play

🎯 Objective

Race against your opponent to solve math problems correctly and fast!

🔗 Connecting

  • Create Game: Generates a 5-letter room code
  • Join Game: Enter your opponent's room code
  • Share the code via text, email, or any messaging app

🎮 Gameplay

  • Answer correctly: +1 point for you
  • Answer first: Lock in your point before opponent!
  • Wrong answer: Opponent gets a chance to score
  • Timeout: No points for either player

📈 Difficulty

  • Correct answers increase your streak
  • 3+ correct in a row = difficulty increase!
  • Higher difficulty = more digits in the problems

⌨️ Keyboard Shortcuts

  • 1, 2, 3, 4: Select answer options
  • Enter: Start match / Play again
  • M: Toggle sound on/off
  • ? Open these instructions

🏆 Scoring

Most points after all questions wins the match!
